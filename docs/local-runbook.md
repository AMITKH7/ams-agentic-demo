# AMS Agentic Ecosystem — Local Runbook

## Purpose

This runbook explains how to run the AMS Agentic Ecosystem demo locally.

The local asset demonstrates an agentic incident-management workflow:

```text
ServiceNow Incident
→ AMS Bridge
→ Atlassian Rovo MCP / Jira / Confluence
→ Optional OpenAI refinement
→ ServiceNow Work Notes
→ Human Gate 1
→ GitHub/Copilot Handoff
→ Copilot Draft PR
→ GitHub PR Webhook
→ ServiceNow Trace Update
→ Local Dashboard / Metrics
